# Machine Coding Questions

A collection of **interview-style React challenges** — color explorer, auth flows, forms, clipboard UX — all implemented as standalone mini-apps and wired together with React Router.

**[→ Live Demo](https://machine-coding-question.netlify.app/)**

---

**Stack:** React 19 · TypeScript · Vite · Tailwind CSS v4 · React Router

```bash
pnpm install && pnpm dev
```
